# Chip Submit

browser extension for torc ambassador content submissions

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** (toggle in top right corner)
3. Click **Load unpacked**
4. Select this project folder
5. The extension icon will appear in your Chrome toolbar

## Setup

1. Click the extension icon
2. Go to the **Settings** tab
3. Enter your email, select your name, and add your ambassador code
4. Click **Save Settings**

## Usage

1. Navigate to any webpage you want to submit
2. Click the extension icon
3. Review the captured URL and your settings
4. Click **Submit** to open the pre-filled Google form
5. Review the form and click Submit
